> Helpers for handling OS environment variables.

It helps to provide:

- user-friendly error messages
- ...

## Installation

Add `:cozy_env` to the list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:cozy_env, <requirement>}
  ]
end
```
